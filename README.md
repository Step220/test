#First project
программа для добавления задачи с описанием, и временем 
функция write_task_to_file() добовляет задачу и время в файл.
функция  show_tasks() для просмотра времени и сравнения с настоящим
main() запускает програму, если файла нет, она создает его 
show_tasks запускается как мультипроценсингом multiproccesing
